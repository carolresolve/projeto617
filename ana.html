Percurso

Já tenho algum contacto com HTML, CSS e JavaScript. Quando comecei a aprender sozinha, sabia apenas o básico, assim como Git e GitHub, que para mim eram apenas ferramentas para fazer commit e enviar ficheiros para o repositório.

Estou a adorar aprofundar mais os meus conhecimentos e perceber melhor como tudo funciona em conjunto.

O que estou a aprender

Durante o curso aprendi desde criar uma pasta pelo terminal até trabalhar com repositórios locais e remotos no Git e GitHub. Também comecei a perceber melhor como funcionam os principais comandos e o fluxo de trabalho em equipa.

  • Commits e histórico de alterações
  • Branches e organização de trabalho em equipa
  • Pull Requests e revisão por colegas

Comandos praticados

Durante o trabalho pratiquei várias coisas importantes com Git e GitHub, como criar branches, gerir alterações e publicar código em equipa.

  • Inicializar repositório git init, ver estado do projeto git status, adicionar mudanças (stage) git add .
  • Criar commits git commit -m "mensagem"
  • Criar branches (“linhas paralelas” de desenvolvimento ) git switch -c nome-da-branch
